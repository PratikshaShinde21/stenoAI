## 📖 About

**StenoAI** is a browser-based transcription platform that lets you record, transcribe, and export audio — all without sending a single byte to any external server. Built by two students who wanted a free, private, and powerful alternative to paid transcription tools.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎙️ **Real-Time Transcription** | Watch your words appear instantly as you speak with high-accuracy speech recognition |
| 🔒 **100% Private** | Everything runs in your browser — no audio or text is ever sent to any server |
| 🌐 **13 Indian Languages** | Hindi, Marathi, Bengali, Tamil, Telugu, Gujarati and more |
| 💾 **Complete Export** | Download transcript, summary and audio recording all in one click |
| ⌨️ **Dual Input Modes** | Live voice recording OR manual typing — whatever works best for you |
| 📊 **Live Statistics** | Track word count, line count, duration and speaker stats in real time |

## 🚀 Getting Started

No installation required! StenoAI runs entirely in the browser.

**1. Clone the repository**
```bash
git clone https://github.com/pratikshashinde2622-droi/stenoai.git
cd stenoai
```

**2. Open in browser**
```bash
# Simply open index.html in any modern browser
open pages/index.html
```

> ✅ Works best on **Google Chrome** or **Microsoft Edge** (Web Speech API support required)

## 🔒 Privacy

StenoAI is built with privacy as the top priority:

- ✅ No backend server
- ✅ No data collection
- ✅ No login required
- ✅ All processing happens locally in your browser
- ✅ Audio never leaves your device
